# Memory Lights — Privacy Policy

**Effective date:** 2026-05-03

**Last updated:** 2026-05-03

---

## Summary

Memory Lights does not collect, store, or transmit any personal data. The app makes no network calls. All game data — high scores, settings, statistics — is stored locally on your device only.

## Data we collect

**None.** We do not collect:

- Personal information (name, email, phone, address)
- Device identifiers (advertising ID, IMEI, MAC, IP)
- Usage analytics or telemetry
- Crash reports
- Location data
- Photos, contacts, calendar, microphone, or camera access
- Any other data of any kind

## Data we share

**None.** We share nothing because we collect nothing.

## Data stored on your device

The app stores the following locally using Android's DataStore (a private app-only preferences mechanism):

- Your high scores (Standard mode and Memory Lights+ mode)
- Your settings (sound pack choice, vibration on/off, sound on/off, difficulty on/off, Memory Lights+ on/off, player timeout, practice mode, reverse mode)
- Your gameplay statistics (games played, average score, total score, best streak, current streak)

This data never leaves your device. It is not backed up to our servers (we do not have servers). You can wipe it any time via the in-app **Reset Score & Statistics** button or by uninstalling the app or clearing app data in Android system settings.

## Permissions

Memory Lights requests one Android permission:

- **`android.permission.VIBRATE`** — used to provide tactile feedback when you press a button. You can disable vibration in the app's top-bar toggle.

The app does not request internet, location, storage, microphone, camera, contacts, or any other permission.

## Network use

The app makes no network calls. There is no connection to any server, no analytics endpoint, no remote configuration, no crash reporting service.

The only outbound action is the optional **Share** button on the game-over screen, which uses the Android share sheet to let you send a score image to other apps you choose. We do not see what you share or where you share it — that is entirely between you, the Android share sheet, and the destination app you pick.

## Children's privacy

Memory Lights is suitable for all ages and does not knowingly collect data from anyone, including children. The app is COPPA and GDPR-K compliant by virtue of collecting nothing.

## Third-party SDKs

**None.** The app's only third-party dependencies are open-source libraries used for game functionality (Jetpack Compose for UI, Koin for dependency injection, Kotlin coroutines for concurrency, Material 3 for design components). None of these libraries make network calls or collect data on the user's behalf.

## Changes to this policy

If a future version of Memory Lights adds any data collection (for example, optional Google Play Games leaderboards), this policy will be updated and the changes will be highlighted in the app's release notes before the change ships. By that point this policy will be revised to describe the new data flows in detail and obtain consent where required.

## Contact
